Place GitHub-facing product screenshots in this folder.

Use these exact filenames:
- `dashboard.png`
- `billing.png`
- `follow-up.png`

Recommended export:
- PNG
- 1440px wide
- light theme only
- browser chrome cropped out unless it materially helps credibility

Blur or redact:
- customer names
- account numbers
- addresses
- phone numbers
- email addresses
- bill IDs
- receipt numbers
- exact balances if they should remain private

Keep these parts sharp:
- page layout
- navigation
- section titles
- tables and cards
- charts or summaries
- buttons and workflow controls

The root repository README is expected to point at this folder once real screenshots are ready.
